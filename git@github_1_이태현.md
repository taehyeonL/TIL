## Fenced Code block

* 코드블록은 backtick 기호 3개를 활용하여 작성

* 코드블록에 특정 언어를 명시하면 신택스 하이라이팅 적용가능
  * ```JAVA```{//주석입니다.)}



## Inline code block 

* 코드 블록은 backtick기호 1개를 인라인에 활용하여 작성
  * At the command prompt, type `nano`.
